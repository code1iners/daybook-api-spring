# daybook-backend-spring
#### Simple Daybook Backend API Server powered by Spring boot



## Api URL
+ [API URL](https://git.heroku.com/radiant-inlet-33352.git)


## Main Tech Stacks
```java
 - java version 11
 - Spring Boot 2.7.2
 - Spring-data-JPA
 - PostgreSQL
 - gradle
```
 
 ## Server Infra
```java
 - Heroku
 - Heroku PostgreSQL
```
 
 ## How To Deploy
 ```linux
 // push to git
 $ git add .
 $ git commit -m "~"
 $ git push origin master
 
 // push to heroku
 $ git push heroku master
 $ heroku open
 ```
